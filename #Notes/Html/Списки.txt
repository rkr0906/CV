Конспект по спискам в HTML
HTML предоставляет два основных типа списков:

Маркированный список (<ul>) – список, в котором каждый элемент отмечен маркером, таким как круг или квадрат. Используется, когда порядок элементов не имеет значения.

Нумерованный список (<ol>) – список, где каждый элемент пронумерован, что используется, когда порядок элементов важен.

Ключевые моменты:
Типы маркеров и нумерации:

Для <ul> можно задавать тип маркера с помощью атрибута type (например, circle, disc, square), но в современных версиях HTML использование этого атрибута не всегда поддерживается напрямую. CSS стили, такие как list-style-type, предпочтительны.
Для <ol> можно задавать тип нумерации с помощью type, например, 1, A, a, I, i.
Вложенные списки:

Списки могут содержать другие списки, что полезно для создания иерархий. Например, внутри <ul> можно добавить <ol>, чтобы создать вложенный упорядоченный список.
Использование Emmet:

Emmet — это инструмент, который помогает писать HTML-код быстрее с помощью сокращений. Например:
ul>li*5 создаст маркированный список с пятью пустыми <li> элементами.
ul>li{Element}*5 создаст маркированный список с пятью <li>, содержащими текст Element.

<ul>
    <li>Элемент 1</li>
    <li>Элемент 2</li>
    <li>Элемент 3</li>
</ul>


<ol>
    <li>Элемент 1</li>
    <li>Элемент 2</li>
    <li>Элемент 3</li>
</ol>


<ul>
    <li>Элемент 1</li>
    <li>Элемент 2
        <ol>
            <li>Вложенный элемент 1</li>
            <li>Вложенный элемент 2</li>
        </ol>
    </li>
    <li>Элемент 3</li>
</ul>
