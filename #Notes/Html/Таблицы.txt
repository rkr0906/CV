HTML предоставляет мощные возможности для представления данных в табличной форме с использованием элемента <table>. Таблицы полезны для структурирования информации, такой как списки, расписания и т.д.

Ключевые элементы таблицы:
Основные теги:

<table>: основной контейнер для создания таблицы.
<thead>: определяет заголовок таблицы, в котором часто указываются названия столбцов.
<tbody>: содержит основные строки данных таблицы.
<tr>: определяет строку таблицы.
<th>: ячейка заголовка (обычно располагается в <thead>).
<td>: ячейка данных (располагается в <tbody>).
Атрибуты:

border: добавляет рамку к таблице и ячейкам (например, border="2").
width: задает ширину таблицы (может быть в пикселях или процентах).
style: позволяет применять CSS-стили. Например, style="text-align: center" центрирует таблицу.
Особенности и рекомендации:

Использование <thead> и <tbody> помогает структурировать таблицу и улучшает читабельность кода.
<th> обычно применяется для создания заголовков столбцов, которые браузеры отображают с полужирным текстом и центрируют по умолчанию.
Для лучшего контроля внешнего вида таблицы рекомендуется использовать CSS вместо устаревшего атрибута border и других визуальных атрибутов.

Примеры использования:
Простая таблица с заголовками и данными:

<table border="2" width="300">
    <thead>
        <tr>
            <th>Name</th>
            <th>Age</th>
            <th>Work position</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Roox</td>
            <td>32</td>
            <td>C# Developer</td>
        </tr>
        <tr>
            <td>Denis</td>
            <td>50</td>
            <td>JS Developer</td>
        </tr>
    </tbody>
</table>


CSS стилизация таблицы:

Вместо атрибутов border и width, таблицу можно стилизовать через CSS для более гибкого контроля:

<style>
    table {
        width: 100%;
        border-collapse: collapse;
    }
    th, td {
        border: 2px solid black;
        padding: 8px;
        text-align: center;
    }
</style>


Секция <caption> для названия таблицы:

Добавляет название таблицы и располагается сразу после тега <table>.

<table>
    <caption>Employee Details</caption>
    <thead>
        <tr>
            <th>Name</th>
            <th>Age</th>
            <th>Work position</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Roox</td>
            <td>32</td>
            <td>C# Developer</td>
        </tr>
    </tbody>
</table>
